# Secret Spots by John Castrillon

##Project MVP
-CRUD app that shows the secret restaurants and bars of NYC

##User Story
-Users will be able to post to a list or view a list of hidden restaurants
in NYC.

##Technologies

-Firebase
-HTML
-CSS
-React Router
